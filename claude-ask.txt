  テーブル一覧

  -- クライアント情報
  clients (
    id: integer PRIMARY KEY,
    name: text,
    fiscal_month: integer,
    staff_id: integer,
    custom_tasks_by_year: json,
    finalized_years: json,
    status: text DEFAULT 'active'
  )

  -- スタッフ情報
  staffs (
    id: integer PRIMARY KEY,
    name: text
  )

  -- 月次タスク
  monthly_tasks (
    id: integer PRIMARY KEY,
    client_id: integer,
    month: text,
    tasks: json,
    status: text,
    url: text,
    memo: text
  )

  -- システム設定
  settings (
    id: integer PRIMARY KEY,
    key: text UNIQUE,
    value: json
  )

  -- 編集セッション管理
  editing_sessions (
    id: integer PRIMARY KEY,
    client_id: integer,
    user_id: text,
    started_at: timestamp,
    expires_at: timestamp
  )

  -- デフォルトタスク
  default_tasks (
    id: integer PRIMARY KEY,
    tasks: json
  )

  🔐 認証・セキュリティ

  Google OAuth認証

  - 認証プロバイダー: Google OAuth 2.0
- 承認済みオリジン: https://jigyousya-final.vercel.app
  - セッション管理: Supabase Auth自動管理
  - ログイン頻度: 1日1回認証（自動セッション管理）

  Row Level Security (RLS)

  - データアクセス制御: Supabase RLS
  - 認証必須: 未ログイン時はデータアクセス不可
  - Anon Key: 公開（フロントエンド用、RLSで保護）

  🎯 主要機能

  1. クライアント管理

  - 一覧表示: 担当者別フィルター、検索機能
  - CRUD操作: 追加、編集、削除、再有効化
  - カスタムタスク設定: クライアント独自の項目設定
  - 年度確定機能: 特定年度のタスク項目ロック

  2. 月次進捗管理

  - 進捗入力: チェックボックス形式
  - ステータス表示: 完了/未完了/一部完了
  - URL・メモ: 各月次記録に付加情報
  - 自動計算: 完了率の自動算出

  3. タスク項目管理

  - デフォルトタスク: 全クライアント共通の初期項目
  - カスタムタスク: クライアント別独自項目
  - 年度継承: 新年度アクセス時の前年度項目継承
  - タスク伝播: 項目変更時の未確定年度への反映

  4. 年度確定システム

  - 確定機能: 特定年度のタスク項目をロック
  - 編集制限: 確定済み年度は編集不可
  - 継承制御: 確定済み項目の新年度継承

  5. データ管理

  - CSVエクスポート: UTF-8 BOM付きExcel対応
  - CSVインポート: 複数エンコーディング対応
  - データベース初期化: 完全リセット機能
  - サンプルデータ: 開発・テスト用データ自動生成

  6. 編集制御

  - 悲観ロック: 編集競合防止
  - 強制解除: 編集セッション強制終了
  - 自動タイムアウト: 一定時間後の自動解除
